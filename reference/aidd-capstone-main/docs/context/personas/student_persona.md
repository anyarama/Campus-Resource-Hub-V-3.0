# Student Persona

## Profile
- **Name:** Alex Chen
- **Role:** Undergraduate Student
- **Department:** Computer Science
- **Year:** Junior (3rd year)

## Goals
1. Find and book study spaces for group projects
2. Access specialized lab equipment for coursework
3. Reserve tutoring sessions for challenging subjects
4. Collaborate with peers on campus

- Difficulty finding available study rooms during exam periods
- Unclear availability for lab equipment
- Need approval for restricted resources
- Want to see resource reviews before booking

- Primarily uses mobile device
- Books resources 1-3 days in advance
- Prefers automatic approval
- Checks availability during breaks between classes
- Values quick search and filtering

## Technical Proficiency
- **Level:** Medium-High
- Comfortable with web applications
- Expects modern, responsive interfaces
- Prefers visual calendars over text listings

## Quotes
> "I need to see what's available right now, not wade through complex menus."

> "Reviews from other students help me know if a study space is actually good."

## Success Metrics
- Can find and book a resource in under 2 minutes
- Receives instant confirmation for non-restricted bookings
- Gets notified when resources become available
